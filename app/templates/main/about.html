About SubwaySSC
Subway Service Corporation System

The Subway Service Corporation System (SSCS) is a full-stack web application built as a Database Management Systems project at SRM Institute of Science and Technology, Chennai.

It addresses real-world challenges in urban metro systems — cumbersome ticketing, lack of operational transparency, and inefficient resource allocation — by providing a comprehensive management platform for both passengers and operators.

Technical Stack
  • Backend: Python Flask with SQLAlchemy ORM
  • Database: MySQL — normalized to 3NF/BCNF
  • Frontend: Custom dark-theme CSS + Bootstrap 5
  • Auth: Flask-Login with Werkzeug password hashing
  • DB Features: Triggers, views, constraints, concurrency control
Project Report

The complete project report covering ER design, relational schema, normalization (1NF–5NF), complex SQL queries, triggers, and concurrency control mechanisms is available in the repository.
